# Best Webpage

## Setup Instructions

Follow these steps to set up and run the project on your local machine.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed. You can verify their installation by running:

```bash
node -v
npm -v
```
## starting
cd best-webpage <br>
npm install <br>
npm run dev <br>
